Assignment 3

Reaction diffusion was an interesting algorithm. Seeing what it did in class was interesting and I loved seeing how it flowed together to make interesting designs. Watching the examples, it was all very relaxing, so that was somewhat of an aesthetic goal I tried to achieve during this. I ensured that the two chemicals would be cool colors, I chose to implement size because I figured that altering the size of the streams would result in some interesting patterns, and I haven't been disappointed. 

Unfortunately, due to finishing this project after class had finished, I was not able to get feedback from a classmate.
