Stack Attack

PreviousNext

Role: Lead Programmer

  • Stack Attack is a game designed for both PC and mobile where the only goal is to stack as many blocks as you can. As the game progresses, the colors of the blocks and background change, intended to produce a somewhat relaxing effect.
  • This game was made for Professor Moriarty's Digital Game Design II class using his custom Perlenspiel engine. It had a prompt of making a game that could be played on both mobile and PC.
  • As the lead programmer, I created the block stacking and the procedural color generation for the blocks and for the background, as they try to contrast as much as possible.
